Map Coloring Game

Color all regions so that no adjacent regions have the same color!

Select a difficulty and start a new game!

How to Play

  1. Select a difficulty level and click "New Game"
  2. Click on a region to select it
  3. Click on a color from the palette to color the selected region
  4. Adjacent regions cannot have the same color
  5. Color all regions to win!
